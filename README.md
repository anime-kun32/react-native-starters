## Getting Started

1. Clone the specific starter you want to use:
```bash
git clone https://github.com/magically-life/react-native-starters.git
```

2. Install dependencies:
```bash
cd <projects/app-name>
npm install
```

3. Start developing:
```bash
npx expo start
```
